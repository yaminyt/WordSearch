# EECE2560 Project - Word Search

This project implements a word search solver using different sorting algorithms.

## Features
- Quicksort for sorting words
- Heapsort using a max heap
- Binary search for word lookup

## Files
- dictionary.cpp / dictionary.h
- heap.cpp / heap.h
- main.cpp

## Description
The program reads a grid and dictionary, sorts the words, and searches for valid words in the grid.
